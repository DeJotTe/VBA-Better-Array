# Capacity

## Get

### Description
Returns the current capacity of the internal array.
### Arguments
None
### Returns
#### (Long)
Current capacity of the internal array.

## Let
### Description
Sets the capacity of the internal array.
### Arguments
#### `Value` (Long) 
The new size of the internal array.
### Returns
None
