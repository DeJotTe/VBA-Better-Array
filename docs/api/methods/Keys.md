# Keys

## Description
The `Keys()` method returns a new array (0 based) that contains the keys for each index in the array.

## Syntax

*expression*.**Keys**()

### Parameters

**None**

### Returns

Type
: `Variant()`

Description
: A 0-based `Variant` array containing the indexes used in the outermost internal array.
