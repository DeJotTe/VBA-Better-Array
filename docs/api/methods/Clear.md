# Clear

## Description
The `Clear()` method clears all entries in the current array but retains the same internal capacity.

## Syntax

*expression*.**Clear**()

### Parameters

**None**

### Returns

Type
: `BetterArray` / `Object`

Description
: The current instance of the BetterArray object with all entries cleared but capacity retained.
