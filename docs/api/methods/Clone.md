# Clone

## Description
The `Clone()` method returns a new BetterArray instance containing the same values as the current instance.

## Syntax

*expression*.**Clone**()

### Parameters

**None**

### Returns

Type
: `BetterArray` / `Object`

Description
: A BetterArray instance containing the same values as the current instance.
