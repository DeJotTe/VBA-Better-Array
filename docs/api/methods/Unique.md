# Unique

## Description
The `Unique()` method removes all duplicate elements from the outermost array such that only unique elements remain.

## Syntax

*expression*.**Unique**() 

### Parameters

**None**

### Returns

Type
: `BetterArray` / `Object`

Description
: The current instance of the BetterArray object with any duplicate elements removed.
