# Transpose

## Description

The `Transpose()` method converts rows to columns and vice versa.

## Syntax

*expression*.**Transpose**()

### Parameters

**None**

### Returns

Type
: `BetterArray` / `Object`

Description
: The current instance with it's stored array transposed.
