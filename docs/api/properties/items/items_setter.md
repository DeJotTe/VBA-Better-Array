# Items Let Accessor

## Description
Stores an array into the BetterArray object

## Syntax

*expression*.**Items** = *Values*

### Parameters

Name 
: `Values`

Type
: `Variant`

Necessity
: Required

Description
: The array of items to store in the BetterArray object instance.


### Returns

**None**
