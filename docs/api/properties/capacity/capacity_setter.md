# Capacity Let Accessor 

## Description
Sets the capacity of the internal array.
## Parameters

*expression*.**Capacity**(*Value*)

---

**Name** 
: `Value`

**Type**
: `Long`

**Necessity**
: Required

**Description**
: The new size of the internal array. 

---

****None****
