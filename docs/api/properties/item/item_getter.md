# Item Get Accessor

## Description
Returns the element stored in the array at the specified index.

## Syntax

*expression*.**Item**(*Index*)

Or

*expression*(*Index*)

### Parameters

Name 
: `Index`

Type
: `Long`

Necessity
: Required

Description
: The index of the element to be returned.

### Returns

Type
: `Variant`

Description
: The element at the specified index.
